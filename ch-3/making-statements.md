## Notes

### Branching with if
``` cpp
if ( test-expression )
{ 
    // statements-to-be-executed-when-true 
}

if ( test-expression )
{
    // statements-to-be-executed-when-true
}
else
{
    // statements-to-be-executed-when-false
}

```

### Switching branches
``` cpp
switch ( variableName )
{
    case value1:
        // do something if value1
        break;
    case value2:
        // do something if value2
        break;
    default:
        // if none of the above values
}
```
